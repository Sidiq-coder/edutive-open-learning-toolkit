Open Source EdTech Toolkit

Fondasi terbuka untuk generate soal, dataset, analisis belajar, dan rekomendasi.

Edutive Open Learning Toolkit adalah proyek open source untuk membangun alur pembelajaran adaptif secara bertahap: mulai dari bank soal terstruktur, validasi dataset, generate soal, analisis hasil latihan, decision support, hingga rekomendasi belajar.
