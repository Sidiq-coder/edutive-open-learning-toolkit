# Use Cases

This document explains possible use cases for Edutive Open Learning Toolkit.

## 1. Question dataset standardization

Developers and teachers can use the dataset format to organize questions with consistent metadata.

This is useful for:

- quiz applications,
- try out platforms,
- pre-test and post-test systems,
- learning analytics experiments,
- and adaptive learning prototypes.

## 2. Pre-test and post-test workflow

A learning platform can use the question dataset to run:

1. pre-test before a topic starts,
2. learning activities,
3. post-test after the topic ends,
4. diagnosis of improvement,
5. and recommendations for follow-up learning.

## 3. Student learning diagnosis

Student answer data can be analyzed to identify:

- weak topics,
- mastered topics,
- common misconceptions,
- difficulty patterns,
- and students who may need support.

## 4. Teacher insight dashboard

A teacher dashboard can use diagnostic output to show:

- class average performance,
- topic-level weaknesses,
- student risk levels,
- remedial groups,
- and recommended teaching focus.

## 5. Decision support for guidance

The diagnostic data can be combined with decision support methods such as:

- SAW,
- TOPSIS,
- Weighted Product,
- or custom weighted scoring.

This can help teachers prioritize students who may need additional guidance.

## 6. Machine learning experiments

In the future, structured learning data can support experiments such as:

- clustering students by learning behavior,
- predicting topic mastery,
- classifying learning risk levels,
- analyzing explanation text,
- and generating recommendation baselines.

## 7. Open education collaboration

The project can become a shared foundation for people who want to contribute:

- sample questions,
- educational metadata,
- diagnostic methods,
- dashboard designs,
- learning analytics notebooks,
- and documentation in different languages.
