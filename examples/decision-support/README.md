# Decision Support Examples

This folder contains fictional example data for decision support utilities.

Files:

- `sample-records.json` — sample records to rank.
- `sample-criteria.json` — criteria weights and criterion types.

The examples use fictional identifiers only.

## Criteria type

A criterion can be:

- `benefit`: higher value is better.
- `cost`: lower value is better.

## Intended use

These files are meant to demonstrate how a topic summary or learning analysis output can be transformed into input for decision support methods such as SAW.

They are not real learner records and should not be treated as real data.
